Difetti identificati durante visual check live:
1. Titoli QGroupBox tagliati in alto (margin-top insufficiente per font 11px)
2. Caratteri grigi (#8b949e su #0d1117) difficili da leggere, contrast ratio ~5:1
3. Pattern panel: "Pattern in Osservazione" troncato + bottone "Pulisci terminati" troppo largo

Fix applicati:

QGroupBox in gui/styles/dark.qss:
- margin-top: 10px -> 18px (spazio sufficiente per titolo)
- padding-top: 8px -> 10px (proporzionale)

- title.padding: 0 5px -> 0 6px (polish)

Palette globale (24 file):
- TEXT_MUTED #8b949e -> #a8b1bb (contrast ratio ~7:1, AAA WCAG)
- TEXT_DIM #484f58 -> #6e7681 (contrast ratio ~3.2:1, leggibile)
- BORDER #30363d invariato

PatternPanel (gui/panels/pattern_panel.py + gui/ui/pattern_panel.ui via i18n):
- Esteso _apply_i18n() per aggiornare _lbl_title e _btn_clear via tr()
- Aggiunte chiavi i18n: pattern.title ("Pattern Attivi" / "Active Patterns"),
  pattern.btn_clear ("Pulisci" / "Clear"), pattern.count_label ("{n} pattern" / "{n} patterns")
